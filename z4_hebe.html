Teoria JavaScript

1.Kiedy powstał JavaScript?
JavaScript został stworzony w 1995 roku przez Brendana Eicha.

2.Dlaczego jest to język skryptowy?
Ponieważ nie jest kompilowany do kodu maszynowego, tylko interpretowany w przeglądarce.

3.Gdzie jest wykonywany JS (klient czy serwer)?
Najczęściej po stronie klienta (przeglądarka), ale także po stronie serwera (np. Node.js).

4.Dlaczego JS to bezpieczne narzędzie?
Ponieważ działa w środowisku przeglądarki i ma ograniczony dostęp do systemu użytkownika.

5.Czy jest to język obiektowy?
Tak, ponieważ obsługuje obiekty, klasy i prototypy.

6.Wymień zalety JS:
Zaleta1 – łatwość nauki
Zaleta2 – działa we wszystkich przeglądarkach
Zaleta3 – duża społeczność i biblioteki

7.Framework to:
Gotowa struktura i zestaw narzędzi do budowy aplikacji.

8.Do aplikacji mobilnych można użyć Framework?
Tak, np. React Native, Ionic.

9.Aplikacje responsywne to:
Takie, które dopasowują się do różnych ekranów i urządzeń.

11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu


window.open(): Jest to wbudowana funkcja przeglądarki, która służy do otwierania nowego okna przeglądarki lub nowej karty.
- toolbar=no → ukrywa pasek narzędzi
- directories=no → ukrywa przyciski katalogów
- menubar=no → ukrywa menu
- height=280 → wysokość okna
- width=160 → szerokość okna
- top=200 → odległość od góry ekranu
- left=200 → odległość od lewej krawędzi ekranu
